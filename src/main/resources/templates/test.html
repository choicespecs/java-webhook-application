curl Reference

Send a webhook using curl:

curl -X POST http://localhost:8080/api/webhook/receive \
  -H "Content-Type: application/json" \
  -d '{
    "eventType": "order.created",
    "eventId": "evt-001",
    "timestamp": "2024-01-01T12:00:00",
    "data": {"orderId": "ORD-123", "amount": 99.99}
  }'
